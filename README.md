## Executar jogo
   **1.** Clone este repositório
```
git clone https://github.com/DanteCampos/jogos-terminal
```
      
   **2.** Acesse a pasta do diretório
```
cd jogos-terminal
```
   **3.** Execute o jogo desejado utilizando o Python3
```
python3 <nome-do-jogo.py>
```

Por exemplo:
```
python3 forca.py
```
